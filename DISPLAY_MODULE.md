# Модуль отображения информации на графике

## 📊 Описание

Модуль `TrendPullbackEA_Display.mqh` добавляет информационную панель на график, которая отображает важную статистику в реальном времени.

---

## ✨ Отображаемая информация

### 1. Информация о счете
- **Баланс** - текущий баланс счета
- **Собственные средства (Equity)** - баланс + незакрытая прибыль/убыток
- **Свободная маржа** - доступная маржа для открытия новых позиций
- **Уровень маржи** - процентное соотношение Equity/Margin
  - ⚠️ Предупреждение при уровне маржи < 100%

### 2. Просадка (Drawdown)
- **Текущая просадка** - абсолютная и процентная от баланса
- **Максимальная просадка** - максимальная просадка от пика equity
- Индикаторы состояния:
  - ✅ Нормальная просадка (< 5%)
  - ⚡ Средняя просадка (5-10%)
  - ⚠️ Высокая просадка (> 10%)

### 3. Открытые позиции
- **Количество** - число открытых позиций этого робота
- **Прибыль** - общая незакрытая прибыль/убыток
- Статус:
  - ✅ В прибыли
  - ❌ В убытке
  - ➖ В безубытке

### 4. Статистика за день
- **Количество сделок** - всего сделок за текущий день
- **Прибыльных** - количество прибыльных сделок
- **Убыточных** - количество убыточных сделок
- **Процент прибыли** - процент прибыльных сделок
- **Прибыль** - общая прибыль/убыток за день
- Статус дня:
  - ✅ Прибыльный день
  - ❌ Убыточный день

### 5. Информация о роботе
- **Символ** - торгуемый инструмент
- **Magic Number** - магический номер робота
- **Время** - текущее время сервера
- **Спред** - текущий спред в пунктах

---

## ⚙️ Параметры настройки

```mql5
input bool   ShowInfoPanel = true;   // Показывать информационную панель
input int    InfoPanelX = 20;        // Позиция X панели (в пикселях)
input int    InfoPanelY = 50;        // Позиция Y панели (в пикселях)
input color  InfoPanelColor = clrDarkSlateGray; // Цвет фона панели
input color  TextColor = clrWhite;  // Цвет текста
```

### Описание параметров:

- **ShowInfoPanel** - включить/выключить отображение панели
- **InfoPanelX** - горизонтальная позиция панели (пока не используется, панель в левом верхнем углу)
- **InfoPanelY** - вертикальная позиция панели (пока не используется)
- **InfoPanelColor** - цвет фона (пока не используется, используется стандартный Comment())
- **TextColor** - цвет текста (пока не используется)

## 🔧 Технические детали

### Обновление информации

- Панель обновляется **на каждом тике** (каждое изменение цены)
- Статистика за день **автоматически сбрасывается** при наступлении нового дня
- Максимальная просадка **отслеживается** от начала работы робота

### Производительность

- Минимальное влияние на производительность
- Использует встроенную функцию `Comment()` для отображения
- Данные берутся из истории сделок только при необходимости

---

## 📊 Отслеживание максимальной просадки

Модуль отслеживает максимальную просадку следующим образом:

1. **При первом запуске** - максимальный equity устанавливается равным текущему equity
2. **При росте equity** - если текущий equity превышает максимальный, обновляется максимум
3. **При достижении нового максимума** - максимальная просадка сбрасывается
4. **При снижении equity** - рассчитывается просадка от максимума и обновляется, если она больше предыдущей

Это позволяет видеть:
- Текущую просадку от баланса
- Максимальную просадку от пика equity (более точный показатель)

---

## ⚠️ Важные замечания

1. **Панель отображается в левом верхнем углу** графика
2. **Информация обновляется в реальном времени** на каждом тике
3. **Статистика за день** сбрасывается автоматически при наступлении нового дня
4. **Максимальная просадка** отслеживается от начала работы робота
5. **При перезапуске робота** максимальная просадка сбрасывается

## 🎯 Использование

### Включение панели:

1. Откройте настройки робота
2. Найдите параметр `ShowInfoPanel`
3. Установите `true` для включения или `false` для выключения
4. Нажмите "OK"

### Просмотр информации:

- Панель автоматически отображается в левом верхнем углу графика
- Информация обновляется автоматически
- Для скрытия панели установите `ShowInfoPanel = false`

---

## 🔄 Интеграция с основным роботом

Модуль автоматически интегрирован в основной файл `TrendPullbackEA.mq5`:

- Подключается через `#include "TrendPullbackEA_Display.mqh"`
- Вызывается в `OnTick()` для обновления информации
- Очищается в `OnDeinit()` при остановке робота
- Инициализируется в `OnInit()` при запуске

---

## 📝 Дополнительные функции

### GetTotalStatistics()

Функция для получения общей статистики за весь период работы робота:

```mql5
string stats = GetTotalStatistics(MagicNumber);
Print(stats);
```

Возвращает:
- Общее количество сделок
- Количество прибыльных/убыточных
- Процент прибыли
- Общую прибыль
- Максимальную прибыль/убыток
