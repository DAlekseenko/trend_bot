# Примеры параметров для TrendPullbackEA

Этот файл содержит готовые наборы параметров для различных стилей торговли.

---

## 🛡️ Консервативная настройка

**Рекомендуется для:** Начинающих трейдеров, небольших счетов, торговли на реальном счете

### Характеристики:
- ✅ Низкий риск на сделку (0.5%)
- ✅ Высокое соотношение риск/прибыль (2.5:1)
- ✅ Больший стоп-лосс для защиты от ложных сигналов
- ✅ Торговля только в определенные часы
- ✅ Меньше сделок, но выше качество

### Параметры:

```mql5
// Основные параметры
RiskPercent      = 0.5   // Риск 0.5% на сделку
EMAPeriod        = 200   // Стандартный период EMA
RSIPeriod        = 14     // Стандартный период RSI
RR               = 2.5    // Высокое соотношение риск/прибыль (2.5:1)
ATRPeriod        = 14     // Стандартный период ATR
ATRMultiplierSL  = 2.0    // Больший стоп-лосс для безопасности
MaxSpreadPoints  = 20     // Строгий контроль спреда
MagicNumber      = 777    // Магический номер

// Дополнительные параметры
UseTimeFilter    = true   // Торговля только в определенное время
StartHour        = 8      // Начало торговли в 8:00
EndHour          = 18     // Конец торговли в 18:00
UseTrailingStop  = true   // Включен трейлинг стоп
TrailingStopATR  = 1.5    // Консервативный трейлинг стоп
TrailingStepATR  = 0.5    // Малый шаг трейлинг стопа
EnableLogging    = true   // Логирование включено
```

### Ожидаемые результаты:
- Количество сделок: 2-5 в неделю
- Процент прибыльных сделок: 60-70%
- Максимальная просадка: низкая
- Рост депозита: медленный, но стабильный

---

## ⚡ Агрессивная настройка

**Рекомендуется для:** Опытных трейдеров, больших счетов, демо-торговли, оптимизации стратегии

### Характеристики:
- ⚠️ Высокий риск на сделку (2%)
- ⚠️ Более низкое соотношение риск/прибыль (1.5:1)
- ⚠️ Меньший стоп-лосс для большего количества сделок
- ⚠️ Круглосуточная торговля
- ⚠️ Больше сделок, но выше риск

### Параметры:

```mql5
// Основные параметры
RiskPercent      = 2.0   // Риск 2% на сделку
EMAPeriod        = 150   // Более чувствительная EMA
RSIPeriod        = 10    // Более чувствительный RSI
RR               = 1.5    // Ниже соотношение риск/прибыль (1.5:1)
ATRPeriod        = 10    // Более быстрый ATR
ATRMultiplierSL  = 1.0    // Меньший стоп-лосс для большего количества сделок
MaxSpreadPoints  = 50     // Более мягкий контроль спреда
MagicNumber      = 777    // Магический номер

// Дополнительные параметры
UseTimeFilter    = false  // Торговля круглосуточно
StartHour        = 0      // Не используется
EndHour          = 23     // Не используется
UseTrailingStop  = true   // Включен трейлинг стоп
TrailingStopATR  = 0.8    // Агрессивный трейлинг стоп
TrailingStepATR  = 0.3    // Малый шаг для быстрого движения стоп-лосса
EnableLogging    = true   // Логирование включено
```

### Ожидаемые результаты:
- Количество сделок: 10-20 в неделю
- Процент прибыльных сделок: 50-60%
- Максимальная просадка: высокая
- Рост депозита: быстрый, но нестабильный

---

## ⚖️ Сбалансированная настройка (РЕКОМЕНДУЕТСЯ)

**Рекомендуется для:** Большинства трейдеров, средних счетов, стабильной торговли

### Характеристики:
- ✅ Умеренный риск на сделку (1%)
- ✅ Стандартное соотношение риск/прибыль (2:1)
- ✅ Баланс между количеством и качеством сделок
- ✅ Гибкость в настройке дополнительных параметров

### Параметры:

```mql5
// Основные параметры
RiskPercent      = 1.0   // Риск 1% на сделку
EMAPeriod        = 200   // Стандартный период EMA
RSIPeriod        = 14     // Стандартный период RSI
RR               = 2.0    // Стандартное соотношение риск/прибыль (2:1)
ATRPeriod        = 14     // Стандартный период ATR
ATRMultiplierSL  = 1.5    // Стандартный стоп-лосс
MaxSpreadPoints  = 30     // Стандартный контроль спреда
MagicNumber      = 777    // Магический номер

// Дополнительные параметры
UseTimeFilter    = false  // Торговля круглосуточно (можно включить при необходимости)
StartHour        = 0      // Не используется
EndHour          = 23     // Не используется
UseTrailingStop  = false  // Трейлинг стоп выключен (можно включить для защиты прибыли)
TrailingStopATR  = 1.0    // Стандартный трейлинг стоп
TrailingStepATR  = 0.5    // Стандартный шаг трейлинг стопа
EnableLogging    = true   // Логирование включено
```

### Ожидаемые результаты:
- Количество сделок: 5-10 в неделю
- Процент прибыльных сделок: 55-65%
- Максимальная просадка: средняя
- Рост депозита: стабильный и предсказуемый

## 🌍 Настройка для торговли в определенное время

**Рекомендуется для:** Трейдеров, которые хотят избежать новостных событий или торговать только в определенные сессии

### Пример 1: Торговля только в европейскую сессию

```mql5
UseTimeFilter    = true
StartHour        = 8      // 8:00 GMT (начало европейской сессии)
EndHour          = 16     // 16:00 GMT (конец европейской сессии)
```

### Пример 2: Торговля только в американскую сессию

```mql5
UseTimeFilter    = true
StartHour        = 13     // 13:00 GMT (начало американской сессии)
EndHour          = 21     // 21:00 GMT (конец американской сессии)
```

### Пример 3: Торговля через полночь (например, 22:00 - 02:00)

```mql5
UseTimeFilter    = true
StartHour        = 22     // 22:00
EndHour          = 2      // 02:00 (следующего дня)
```

## 📊 Настройка для разных инструментов

### Для валютных пар (EURUSD, GBPUSD и т.д.)

```mql5
RiskPercent      = 1.0
EMAPeriod        = 200
RSIPeriod        = 14
RR               = 2.0
ATRPeriod        = 14
ATRMultiplierSL  = 1.5
MaxSpreadPoints  = 30     // Для основных пар обычно достаточно 30 пунктов
```

### Для индексов (SPX500, NAS100 и т.д.)

```mql5
RiskPercent      = 1.0
EMAPeriod        = 200
RSIPeriod        = 14
RR               = 2.0
ATRPeriod        = 14
ATRMultiplierSL  = 2.0    // Больший стоп-лосс для индексов
MaxSpreadPoints  = 50     // Для индексов спред может быть больше
```

### Для криптовалют (BTCUSD, ETHUSD и т.д.)

```mql5
RiskPercent      = 0.5    // Меньший риск из-за высокой волатильности
EMAPeriod        = 200
RSIPeriod        = 14
RR               = 2.5    // Выше соотношение риск/прибыль
ATRPeriod        = 14
ATRMultiplierSL  = 2.5    // Больший стоп-лосс для криптовалют
MaxSpreadPoints  = 100    // Для криптовалют спред может быть очень большим
```

## 🔧 Настройка трейлинг стопа

### Консервативный трейлинг стоп (защита прибыли)

```mql5
UseTrailingStop  = true
TrailingStopATR  = 1.5    // Большее расстояние до стоп-лосса
TrailingStepATR  = 0.5    // Малый шаг - стоп-лосс двигается медленно
```

### Агрессивный трейлинг стоп (быстрая защита)

```mql5
UseTrailingStop  = true
TrailingStopATR  = 0.8    // Меньшее расстояние до стоп-лосса
TrailingStepATR  = 0.3    // Малый шаг - стоп-лосс двигается быстро
```

### Сбалансированный трейлинг стоп

```mql5
UseTrailingStop  = true
TrailingStopATR  = 1.0    // Стандартное расстояние
TrailingStepATR  = 0.5    // Стандартный шаг
```

## 💡 Советы по настройке

1. **Начните с сбалансированной настройки** - это безопасный старт
2. **Тестируйте на демо-счете** минимум 1-2 месяца перед использованием на реальном счете
3. **Адаптируйте под инструмент** - разные инструменты требуют разных параметров
4. **Используйте фильтр времени** для избежания новостных событий
5. **Включайте трейлинг стоп** только после тестирования базовой стратегии
6. **Мониторьте логи** для понимания работы робота
7. **Не увеличивайте риск** без тщательного тестирования

---

## ⚠️ Важные предупреждения

- Всегда тестируйте новые параметры на демо-счете
- Не используйте агрессивные настройки на реальном счете без опыта
- Учитывайте волатильность инструмента при выборе параметров
- Регулярно проверяйте логи на наличие ошибок
- Адаптируйте параметры под ваш стиль торговли и размер счета
