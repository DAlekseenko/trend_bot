# Инструкция по установке TrendPullbackEA

## 📦 Быстрая установка

### Вариант 1: Установка в папку Experts (рекомендуется)

1. Скопируйте **все файлы** из этой папки в папку `MQL5/Experts/TrendPullbackEA/` вашего MetaTrader 5
   ```
   C:\Users\ВашеИмя\AppData\Roaming\MetaQuotes\Terminal\XXXXX\MQL5\Experts\TrendPullbackEA\
   ```

2. Откройте MetaEditor (F4 в MT5 или через меню Tools → MetaQuotes Language Editor)

3. Откройте файл `TrendPullbackEA.mq5` в MetaEditor

4. Нажмите F7 для компиляции или кнопку "Compile"

5. Убедитесь, что компиляция прошла успешно (внизу должно быть "0 error(s), 0 warning(s)")

6. Перетащите робота из папки "Navigator" → "Expert Advisors" → "TrendPullbackEA" на график нужного инструмента

### Вариант 2: Установка в Include (для продвинутых пользователей)

Если вы хотите использовать модули в других роботах:

1. Скопируйте файлы `.mqh` в папку `MQL5/Include/`
2. Скопируйте `TrendPullbackEA.mq5` в папку `MQL5/Experts/`
3. В основном файле измените пути включения на:
   ```mql5
   #include <TrendPullbackEA_Inputs.mqh>
   #include <TrendPullbackEA_Types.mqh>
   // и т.д.
   ```

## ✅ Проверка установки

После установки проверьте:

1. ✅ Все файлы находятся в одной папке
2. ✅ Файл `TrendPullbackEA.mq5` компилируется без ошибок
3. ✅ Робот появляется в списке Expert Advisors
4. ✅ При перетаскивании на график открывается окно настроек

## 🔧 Настройка параметров

### Первый запуск:

1. Откройте настройки робота (двойной клик на роботе на графике)
2. Выберите один из готовых наборов параметров из файла `PARAMETERS_EXAMPLES.md`:
   - **Консервативная** - для начинающих
   - **Сбалансированная** - для большинства (рекомендуется)
   - **Агрессивная** - для опытных трейдеров
3. Скопируйте параметры в настройки робота
4. Нажмите "OK"

### Важные параметры для первого запуска:

- `RiskPercent = 1.0` - начните с 1% риска
- `EnableLogging = true` - включите логирование для отслеживания работы
- `UseTimeFilter = false` - можно оставить выключенным для начала
- `UseTrailingStop = false` - можно включить после тестирования

## 🐛 Решение проблем

### Робот не компилируется:

- Убедитесь, что все файлы `.mqh` находятся в той же папке, что и `.mq5`
- Проверьте, что пути включения правильные (без `<` и `>` для локальных файлов)
- Убедитесь, что нет синтаксических ошибок

### Робот не появляется в списке:

- Перезапустите MetaTrader 5
- Проверьте, что файл находится в правильной папке `MQL5/Experts/`
- Убедитесь, что компиляция прошла успешно

### Робот не открывает позиции:

- Проверьте логи в разделе "Эксперты"
- Убедитесь, что автоматическая торговля разрешена (кнопка "AutoTrading" вверху терминала)
- Проверьте параметры спреда (`MaxSpreadPoints`)
- Убедитесь, что условия входа выполнены (тренд + RSI)
